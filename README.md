# 🎙️ VoiceViber – Voice-Based AI Coding Assistant

VoiceViber is a voice-driven AI coding agent that enables developers to build applications using simple natural language commands. It combines speech recognition, large language models (LLMs), and tool execution to automate development workflows.

---

## 🚀 Features

- 🎤 **Voice-to-Code Generation**  
  Convert spoken instructions into functional code in real-time

- 🧠 **LLM-Powered Understanding**  
  Uses advanced language models to interpret developer intent accurately

- 🔧 **Tool Calling & Local Execution**  
  Executes commands directly on the user’s local system to automate tasks

- 🔄 **Context-Aware Workflow**  
  Maintains session context for multi-step development processes

- 🛡️ **Error Handling & Validation**  
  Handles ambiguous inputs, noise, and unsafe commands effectively

---

## 🏗️ Tech Stack

- **Languages:** Python  
- **AI/ML:** LLMs, Prompt Engineering  
- **Speech Processing:** Speech-to-Text APIs  
- **Frameworks:** LangChain / LangGraph
- **Execution Layer:** Local system command execution  


---

## ⚙️ How It Works

1. User gives a voice command  
2. Speech is converted to text using STT  
3. LLM processes intent and generates action/code  
4. Agent decides whether to:
   - Generate code  
   - Execute a command (tool call)  
5. Output is returned or executed locally
